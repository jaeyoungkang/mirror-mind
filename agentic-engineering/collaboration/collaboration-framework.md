# AI-인간 협업 체계 (Collaboration Framework)

## 핵심 전제
- AI는 도구가 아닌 독립된 행위 주체(Co-actor)
- 체계는 사전 설계가 아닌 실전에서 점진적으로 구축

## 작동한 패턴
1. 원칙 우선 — 일하기 전에 "어떻게 일할 것인가"를 먼저 합의
2. 단일 원본(SSoT) — 원칙은 한 곳에서만 관리, 나머지는 참조
3. 점진적 확장 — 필요가 체감될 때 구조를 추가
4. 합의 기반 수정 — 핵심 문서는 반드시 합의 후 변경
5. 선제적 제안 — AI가 지시를 기다리지 않고 먼저 제안

## 구조 설계
- 철학/원칙 문서 → 도메인별 원칙 문서 → 실행 파일(tasks/)
- 각 계층이 상위를 참조하는 트리 구조
